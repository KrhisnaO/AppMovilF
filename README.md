# AppMovilF


Aplicación Android desarrollada en Android Studio, orientada a mejorar la experiencia de uso y accesibilidad para personas con discapacidad sensorial.

La aplicación permite a los usuarios registrarse, iniciar sesión y recuperar su contraseña mediante una interfaz simple, clara y accesible.

---

## Funcionalidades

- Pantalla de inicio con navegación principal
- Inicio de sesión con validación de usuario
- Registro de usuarios con validaciones básicas
- Recuperación de contraseña por correo electrónico
- Navegación entre vistas con Jetpack Compose

---

## Tecnologías

- Android Studio
- Kotlin
- Jetpack Compose
- Material Design
- Navigation Compose
